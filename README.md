# Letos
A free, open source, multi-platform SQLite database manager written in C++, with use of Qt framework.

### Formerly SQLiteStudio
This project was previously known as SQLiteStudio. The name was changed to avoid confusion with the official SQLite project, as SQLiteStudio is an independent application not affiliated with the SQLite authors.
